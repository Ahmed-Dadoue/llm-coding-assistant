# LLM Coding Assistant

Dieses Projekt ist eine Web-Anwendung, die mit Streamlit entwickelt wurde.

Die App funktioniert als KI-basierter Coding Assistant.  
Benutzer können Fragen stellen oder Code eingeben und erhalten passende Erklärungen oder Vorschläge.

## Verwendete Technologien
- Python
- Streamlit
- OpenAI API
- GitHub
- Streamlit Cloud (Deployment)
